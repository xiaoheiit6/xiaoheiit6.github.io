## 一、启动/重启/停止Nginx命令

- Debian系

  ```bash
  系统：Debian
  启动命令：service nginx start
  重启命令：service nginx restart
  停止命令：service nginx stop
  ```

- CentOS

  ```bash
  系统：CentOS
  启动命令：systemctl start nginx
  重启命令：systemctl restart nginx
  停止命令：systemctl stop nginx
  ```
